The program only has one python dependency called 'pythonnet' which is the library that allows us to call dotnet assemblies directly from python. It will also require that Atlas 10 is installed in the default location and will need to be run using the 64-bit version of Python.

It shows how to use filters to find sessions that you are interested in, load those sessions, position the read cursor to a desired lap in the session and the extract certain parameters. It is setup initially to load 20 samples from the start of the second lap for vCar and gLat for each race session in a specified date range.
