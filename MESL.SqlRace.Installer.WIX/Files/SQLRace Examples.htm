SQL Race examples

The SQL Race examples demonstrate how the SQL Race APIs can be used. They are written in C# .NET and require Microsoft Visual Studio 2013 or above to compile/run.

The examples can be found within the installation directory, which will typically be C:\Program Files\McLaren Electronic Systems\SQL Race\Examples.

Click here to open the examples.

 

List of examples (C# .NET)

Name Description
MESL.SqlRace.Examples.Recorder MESL.SqlRace.Examples.Recorder.UI Demonstrates how a custom recorder can be developed for SQL Race Server. This example has 2 parts, the UI assembly and recorder assembly. When compiled and copied to the McLaren Electronic Systems\SQL Race\Bin\Extensions directory, this recorder can be loaded and example data is written to SQL Race.
MESL.SqlRace.Examples.Sessions.CSharp Demonstrates general operations with the SQL Race API such as creating sessions, adding data/laps/events, loading a saved session and searching sessions by session items etc.
MESL.SqlRace.Examples.ReadExcelData Demonstrates reading data from an excel sheet (such as session constant) and adding it to the SQL Race session.
MESL.SqlRace.Examples.DataDistribution Demonstrates the usage of the Data Distribution feature (distribution between a local SQL Race database and a centralized SQL Race database server).
